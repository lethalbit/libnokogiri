enum libnokogiri::pcap::pcap_variant_t¶

Types of possible pcap formats.

Values:

enumerator Standard¶
enumerator Modified¶

“Standard” pcap file magic

enumerator IXIAHW¶

Magic for the modified pcap format introduced by Alexey Kuznetsov’s patches to some versions of libpcap

enumerator IXIASW¶

Magic for IXIA’s lcap format

enumerator Nanosecond¶

Magic for IXIA’s lcap format

enumerator SwappedStantard¶

Magic for the modified pcap format introduced by Ulf Lamping’s patches. It’s identical to the standard but the timestamps are in nanoseconds. Swapped magic for standard pcap files

enumerator SwappedModified¶

Swapped magic for modified pcap files

enumerator SwappedIXIAHW¶

Swapped magic for IXIA’s lcap

enumerator SwappedIXIASW¶

Swapped magic for IXIA’s lcap

enumerator SwappedNanosecond¶

Swapped magic for nanosecond pcap files
