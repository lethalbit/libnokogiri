Namespace libnokogiri::pcapng::blocks
=====================================


pcapng blocks that are defined as per the standard 
 


.. contents:: Contents
   :local:
   :backlinks: none




Detailed Description
--------------------

This namespace contains all the pre-specified blocks as defined by the pcapng RFC.
For documentation on how to add new blocks to libnokogiri see the ``Adding Blocks`` section in ``Extending libnokogiri``
